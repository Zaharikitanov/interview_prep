# QA Interview Questions

## Най-чести интервю въпроси (Fundamentals)

### 1. Основи на QA и тестването

**Защо тестваме даден софтуер?**

**Каква е разликата между QA, QC и Testing?**

**Каква е разликата между test case и test scenario?**

**Какво е happy path? Какво е negative testing?**

**Какво е SDLC & STLC и кои са фазите?**

**Каква е разликата между Verification и Validation?**

**Какво е Bug Lifecycle?**

**Каква е разликата между Severity и Priority? Дай пример.**

**Какво е Regression testing и кога се прави?**

**Какво е Smoke vs Sanity testing?**

**Каква е разликата між re-test и regression?**

### 2. Test Design техники (основни)

**Обясни Equivalence Partitioning с пример**

**Обясни Boundary Value Analysis с пример**

### 3. Bug reporting

**Какво прави един добър bug report?**

**Кои полета винаги попълваш?**

**Какво правиш, ако dev каже, че не е бъг?**

**Как приоритизираш бъгове, ако имаш 20 открити?**

**Да опише бъг с всички необходими неща - Бъгът е, че Logout на Mobile/App крашва като натиснеш бутона**

### 4. Практически казусни въпроси (common)

**Имаш форма за логин. Какви тестове би написал?**

**Как би тествал календар?**

**Как би тествал онлайн магазин?**

### 5. Инструменти и процеси

**С какви bug tracking systems си работил?**

**Работил ли си по Agile / Scrum проект?**

### 6. API & HTTP основи

**Какво е HTTP status code 200 / 400 / 401 / 403 / 500?**

**Какво е разликата между GET, POST, PUT, DELETE?**

### 7. Малки „хитри" въпроси (основни)

**Може ли софтуер да е 100% тестван?**
